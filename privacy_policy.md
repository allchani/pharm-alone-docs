# 〈나홀로 약국〉 개인정보처리방침

〈나홀로 약국〉(이하 '본 앱')은 사용자의 개인정보 보호를 매우 중요하게 생각하며,
관련 법령을 준수하고 있습니다. 본 개인정보처리방침은 앱 사용 시 개인정보가 어떻게
취급되는지 설명합니다.

본 앱은 두 가지 사용 모드를 제공합니다. 모드에 따라 수집·저장·전송되는 정보가 달라집니다.

- **오프라인 모드** (혼자 사용): 회원가입 없이 본인 기기에서만 사용.
- **온라인 모드** (약국 멤버): 회원가입 후 다른 동료와 약품 정보를 공유.

> **중요**: 오프라인 모드에서는 어떠한 정보도 외부로 전송되지 않으며,
> 아래 §3 (수집 정보) 의 모든 항목은 **오직 온라인 모드에서만** 해당됩니다.

---

## 1. 오프라인 모드 — 정보 비수집

본 앱을 회원가입 없이 사용하는 **오프라인 모드** 에서는:

- 사용자를 식별할 수 있는 어떠한 개인정보(이름·전화번호·이메일 등)도 **수집하지 않습니다**.
- 입력하신 모든 약품 데이터 및 수납장 정보는 사용자의 스마트폰 내부(로컬 데이터베이스)
  에만 저장됩니다.
- **외부 서버 전송이 일절 없습니다**. 개발자나 제3자가 해당 데이터에 접근할 수 없습니다.
- 앱을 삭제하면 기기 내부의 모든 데이터가 함께 즉시 파기됩니다.

오프라인 모드만 사용하시는 경우, 본 정책의 §3 이하 (온라인 모드 관련 사항) 는 해당되지
않습니다.

---

## 2. 기기 권한 사용 안내

### 2-a. 카메라 권한

본 앱은 다음 목적으로 기기의 '카메라 권한' 을 요청·사용합니다.

- **의약품 2차원 바코드 인식** — 재고 입력 자동화.
- **수납장 사진 촬영** — 수납장 위치를 사진으로 기억 (선택 기능).
- **알약 카운팅** — 카메라로 알약 갯수 자동 인식 (선택 기능).

처리 방식:

- 카메라는 오직 위 기능을 사용하는 순간에만 작동합니다.
- 바코드 스캔 / 알약 카운팅 시 촬영된 이미지는 기기 내부에 저장되지 않으며 외부로
  전송되지 않습니다.
- 수납장 사진은 사용자가 명시적으로 "사진 등록" 을 선택한 경우에만 저장됩니다.
  - 오프라인 모드: 기기 내부에만 저장.
  - 온라인 모드: 약국 멤버와 공유하기 위해 클라우드에 저장 (§3-c 참조).

### 2-b. 사진 라이브러리 권한 (선택 사용 시)

수납장 사진을 갤러리에서 선택할 때만 사용합니다. 다른 사진은 접근하지 않습니다.

## 3. 온라인 모드 — 수집·저장 정보

회원가입 후 **온라인 모드** 에서는 다음 정보가 수집·저장됩니다. 회원가입 자체가
선택이며, 가입하지 않으면 §1 (오프라인 모드) 와 같이 정보가 수집되지 않습니다.

### 3-a. 수집 항목

| 항목 | 수집 시점 | 용도 |
|---|---|---|
| 이메일 주소 | 회원가입 시 | 계정 식별 / 로그인 / OTP 인증 코드 발송 |
| 비밀번호 (해시) | 회원가입 시 | 로그인 인증 (원본 비밀번호는 저장하지 않습니다) |
| 닉네임 | 약국 합류·생성 시 | 동료에게 표시될 이름 |
| 약품 데이터 | 사용 중 | 재고 정보 (약품명·수량·유효기간·로트번호·바코드) |
| 수납장 정보·사진 | 사용 중 | 약품 위치 표시 |
| 동기화 메타데이터 | 자동 | 동시 편집 충돌 처리 / 변경 이력 |
| 진단 로그 | 자동 (앱 오류 발생 시) | 앱 안정성 개선 — 영문 에러 메시지 (PII 미포함), 앱 버전, 플랫폼 종류 |

본 앱은 이름, 전화번호, 주소, 주민등록번호, 신용카드 정보 등 위 표 외의 개인정보를
수집하지 않습니다.

진단 로그는 사전 검토를 거쳐 식별 가능한 정보 (이메일, 닉네임, 약품 데이터 등) 가
포함되지 않도록 설계되어 있으며, 앱 동작 중 발생한 일부 오류 메시지만 기록됩니다.

### 3-b. 약품 데이터의 성격

본 앱이 저장하는 "약품 데이터" 는 약국이 보유한 의약품의 일반 정보 (제품명·바코드·
유효기간 등) 일 뿐이며, 환자나 처방 정보는 일절 포함하지 않습니다.

### 3-c. 저장 위치

- 클라우드 저장 서비스: **Supabase** 의 **한국 (서울) region**.
- 데이터는 **국내 서버에 저장 처리**됩니다 (개인정보 보호법 제28조의2 의 국외 이전에
  해당하지 않습니다).
- 모든 데이터는 **HTTPS / TLS 암호화** 로 전송되며, 서버 측에도 암호화 저장됩니다.
- 약국 단위로 데이터가 격리되어 다른 약국 사용자가 접근할 수 없습니다 (Row-Level
  Security).

### 3-d. 보유 및 이용 기간

| 액션 | 처리 |
|---|---|
| 회원탈퇴 | 계정 정보 즉시 영구 파기 |
| 약국 자체 삭제 | 마킹 후 **5분 retention** 뒤 약국 데이터 영구 cascade 삭제 (실수 복구 가능 시간) |
| 약국 떠나기 | 본인의 멤버십 정보만 비활성화. 약국 / 다른 멤버 데이터는 영향 없음 |
| 약국 데이터 초기화 | 약국 내 약품·수납장·사진·로그 즉시 파기 |
| 멤버 비활성화 | 해당 멤버의 멤버십 정보만 비활성화 |
| 진단 로그 | **30일** 후 자동 삭제 |

회원탈퇴 시 본인의 모든 개인정보가 즉시 파기되며, 백업·로그에서도 제거됩니다.

## 4. 제3자 제공 및 처리 위탁

본 앱은 사용자 데이터를 **광고, 마케팅, 분석, 판매 등 어떤 목적으로도 제3자에게
제공하지 않습니다**.

다만, 온라인 모드의 핵심 기능 (인증·동기화·저장) 을 위해 다음 처리위탁자에게 일부
정보가 처리됩니다.

| 위탁 받는 자 | 위탁 업무 | 처리 정보 | 데이터 위치 |
|---|---|---|---|
| Supabase, Inc. | 인증 / 데이터베이스 / 파일 저장 / 진단 로그 보관 | 이메일·비밀번호 해시·닉네임·약품 데이터·수납장 사진·진단 로그 | 한국 (서울) region |

처리위탁 계약은 사용자의 개인정보 보호를 보장하는 조건으로 체결됩니다. 본 앱은 사용자
데이터를 **국외로 이전하지 않습니다**.

## 5. 사용자의 권리

사용자는 언제든지 다음 권리를 행사할 수 있습니다.

- **자신의 정보 열람**: 앱의 [설정] 메뉴에서 본인의 계정 / 약국 정보를 확인 가능.
- **수정 요청**: 닉네임 등 수정 가능 항목은 앱 내에서 직접 수정.
- **삭제 요청**: 앱의 [설정 → 회원탈퇴] 로 즉시 처리. 또는 §6 의 연락처로 요청.
- **이의 제기**: §6 연락처 또는 한국인터넷진흥원(KISA) 개인정보침해신고센터 (privacy.kisa.or.kr / ☎️ 118).

## 부록 — 기존 사용자 안내

이전 버전 〈나홀로 약국〉(오프라인 전용) 을 사용해 오신 사용자께:

- 업데이트 후에도 **기존 데이터는 그대로 유지**됩니다.
- 회원가입은 선택입니다. 가입하지 않으시면 기존처럼 **오프라인 모드** 로 정보 수집 없이
  사용하실 수 있습니다.
- 동료와 약품 정보를 공유하고 싶으시면 회원가입 → 약국 만들기 / 합류하기 로 온라인
  모드 진입.
- 오프라인 모드의 데이터를 약국에 합류한 뒤 가져오는 기능 (Solo → Online import) 도
  제공됩니다. [user_manual.md §13-c](./user_manual.md#13-c-데이터-가져오기-owner-전용) 참조.
